# 🏢 Lumina Corporate | Site Institucional Premium

![Preview do Projeto]

## 📖 Sobre o Projeto

Este é um template de site institucional focado no setor corporativo (consultorias, assessorias, escritórios de advocacia, etc.). O projeto foi desenvolvido com o objetivo de transmitir **credibilidade, modernidade e confiança**, características essenciais para empresas do segmento B2B (Business to Business).

O layout foi construído do zero, sem o uso de frameworks externos, demonstrando domínio sólido sobre as tecnologias fundamentais da web.

## ✨ Funcionalidades

- **Design 100% Responsivo:** Adapta-se perfeitamente a dispositivos móveis, tablets e desktops.
- **Animações "Scroll Reveal":** Os elementos surgem suavemente na tela à medida que o usuário rola a página, criando uma experiência fluida e premium.
- **Variáveis CSS (Custom Properties):** Facilita a manutenção e a alteração do tema de cores do site em um único lugar.
- **Header Fixo (Sticky):** Navegação sempre acessível no topo da página.
- **Semântica Web:** Uso adequado de tags HTML5 (header, section, footer, nav) para melhor SEO e acessibilidade.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização avançada utilizando **Flexbox** e **CSS Grid**, Media Queries para responsividade e Variáveis CSS.
- **JavaScript (Vanilla):** Lógica simples e eficiente para ativar as animações de rolagem (Scroll Reveal).
- **Google Fonts:** Tipografia elegante utilizando `Montserrat` (Títulos) e `Open Sans` (Textos).
